<h1 align="center">Installation</h1>

Install all dependencies using:
```sh
yarn add package.json
```
Compile source code:
```sh
yarn build
```
Create `tokens.json` file inside `files` directory, and put this code into this file:
```json
{
    "discord": "discord.bot.token"
}
```
Start bot:
```sh
yarn start
```
